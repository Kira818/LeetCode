# 题目
给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串，判断字符串是否有效。

有效字符串需满足：

左括号必须用相同类型的右括号闭合。
左括号必须以正确的顺序闭合。
注意空字符串可被认为是有效字符串。

# 示例
```
输入: "()"
输出: true

输入: "()[]{}"
输出: true

输入: "(]"
输出: false

输入: "([)]"
输出: false

输入: "{[]}"
输出: true
```

# 解法
1、以“)”、“]”、“}”为 key 建立哈希表；  
2、迭代字符串；  
3、读到“(”、“[”、“{”入栈；  
4、读到“)”、“]”、“}”出栈并对比哈希表；  
5、最后判断栈是否为空。
