# 题目
插入排序算法：

    1.插入排序是迭代的，每次只移动一个元素，直到所有元素可以形成一个有序的输出列表。
    2.每次迭代中，插入排序只从输入数据中移除一个待排序的元素，找到它在序列中适当的位置，并将其插入。
    3.重复直到所有输入数据插入完为止。
    
# 示例
```
输入: 4->2->1->3
输出: 1->2->3->4

输入: -1->5->3->4->0
输出: -1->0->3->4->5
```

# 解法
单向链表表内排序：  
1、定义 res := &ListNode{0, nil}；  
2、迭代链表；  
3、从第二个开始，判断是否小于前一个；  
4、如果小于，则当前节点从 res.Next 开始迭代，比较大小；  
5、如果不大于某个节点，则插入该节点前；  
6、原节点指向前节点，跳出内层循环；  
7、返回 res.Next。
